#  Uso de Inteligencia Artificial en el Proyecto

## Herramientas utilizadas

Se utilizó **Claude (Anthropic)** como asistente de IA durante el desarrollo de este proyecto.

## Contribuciones de la IA

### 1. Estructura del proyecto
- Generación de la estructura de carpetas y archivos base.
- Configuración inicial del repositorio Git.

### 2. HTML
- Maquetación del `index.html` con semántica correcta.
- Formulario de ingreso de tareas con atributos de accesibilidad.
- Estructura del tablero Kanban en 3 columnas.

### 3. CSS
- Sistema de variables CSS para colores, tipografía y espaciado.
- Estilos para tarjetas, formulario, header y footer.
- Media queries para 3 breakpoints (1024px, 768px, 480px).
- Animaciones de entrada y efectos hover.

### 4. JavaScript
- Lógica de agregar, cambiar estado y eliminar tareas.
- Actualización dinámica de contadores.
- Validación de formulario.
- Animación de eliminación con transición CSS.

## Proceso de trabajo

1. Se proporcionó el enunciado del proyecto como instrucción.
2. La IA generó todos los archivos siguiendo las especificaciones.
3. Se revisó el código para asegurar el cumplimiento del checklist.

## Aprendizajes

El uso de IA permitió acelerar el desarrollo inicial, pero requirió comprensión del código generado para ajustarlo a los requisitos específicos del proyecto.
